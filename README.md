\# Battery Management System Simulation



\## Overview

This project simulates a battery management system (BMS) with protection logic including undervoltage and overcurrent protection.



\## Features

\- SOC-based energy modeling

\- Undervoltage cutoff

\- Overcurrent protection

\- Internal resistance (IR drop)



\## Results

\- Higher current → faster discharge

\- IR reduces terminal voltage

\- Protection prevents unsafe operation



\## How to Run

python battery\_sim.py
